# 📊 Vignan ERP Dataset Repositories

This directory has been organized into structured profiles to easily swap between academic configurations:

## 📁 `demo-data/rich/`
* **Purpose**: Comprehensive, high-density simulation covering **all 11 departments** (CSE, IT, ECE, EEE, CIVIL, AI&DS, AIML, CSBS, MECH, CHEMICAL, BIOTECH).
* **Contents**: Mapped students, subjects, 15 days of attendance streams (averaging ~82%), and detailed scores.
* **Usage**: Best for institutional visual demonstrations.

## 📁 `demo-data/aids/`
* **Purpose**: Targeted profile focusing heavily on the **AI&DS (AIDS)** department.
* **Contents**: 15 custom AI&DS students, 4 AI&DS professors, 4 courses, high **88% present rate** to keep AI&DS in a leading standing, and scores averaging **76%** (placing AI&DS beautifully in 5th place).
* **Usage**: Ideal for target testing specialized department statistics.

## 📁 `demo-data/base/`
* **Purpose**: Initial reference dataset configurations for system backups.
